# How to use Taskwarrior to apply GTD

## Tags

- `+in` = inbox
- `+wating` = waiting list
- `+next` = next action list

## commands

- `in` = `task add +in`
- `tickle` = `task add +in wait:$deadline`
- `think` = `tickle +1d`
