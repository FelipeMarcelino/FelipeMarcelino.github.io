# Task Definition
#tasks #organization #howtodo #system #ideas

Have a defined task is more important than have willpower to execute the task. Willpower is a limited energy and will
vanish sometime.
- Try to define the task as more clear way possible
- Break tasks on small steps
- Make a workflow with everything you need to know to execute the task.
- Good solution for tasks are simple
- It is important to have a structure and system for every task
- Don't be [[hsyx-dont-be-multitask]]#, people can only focus on one thing at time.
